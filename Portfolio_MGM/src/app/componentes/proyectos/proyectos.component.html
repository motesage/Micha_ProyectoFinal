Sistema de control de tiempo y asistencia

TimeTech SQL

Pensado para responder a las mayores exigencias y con manejo completo de las variables biométricas de identificación TimetechSQL es la solución en uso en las mayores instalaciones de control de asistencia del país. Procesamiento y lectura distribuidos. Optimizado para instalaciones con gran cantidad de usuarios y personal. Análisis inteligente de fichadas. Ilimitada cantidad de horarios, simples, personales, rotativos y complejos. Flexibilidad en el motor de base de datos (MSSQL, MySQL, Oracle). Completo registro de auditoría. Módulo completo de seguridad informática según requisitos del BCNA.
